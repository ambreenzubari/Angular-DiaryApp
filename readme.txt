	-----Diary App----------

this project is a diary app which has information of contacts, user can login in the app and can save his contacts. in this projets there are many components include:
	add-contact: this will help to add the contacts in the firestore
	anav-bar: this include options i.e add contact, list contact and log out
	home-component: this include functionality of signIn and create account
	list-contacts: this will show the list of contacts
It includes a gaurd
It has folder called Model, which includes a phoneNumber Model and a service class.
